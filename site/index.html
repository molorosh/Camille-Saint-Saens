…some neat CSS tricks!

This is kind of like a mood-board of (mostly) other people's CSS experiments.

The header uses a flip-card animation that uses the CSS perspective property and also some on-hover animation. It's obviously a great fit for rectangles (using the aspect-ratio property) but other shapes are possible in CSS too!

The progress bar uses scroll-based animations to indicate the user's position on the page. q.v. https://css-tricks.com/unleash-the-power-of-scroll-driven-animations/

The sibling-index allows for a lot of creative possibilities in CSS layouts. Especially since we can do a lot of calculations for postion and colour now. q.v. https://css-tricks.com/almanac/functions/s/sibling-index/
